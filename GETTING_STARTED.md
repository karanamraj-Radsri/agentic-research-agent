# Getting Started Guide

Welcome to the Agentic Research Agent! This guide will help you set up and start using the application in no time. Below are the sections included in this guide:

## Quick Start in 5 Minutes
1. **Clone the repository**:  
   `git clone https://github.com/karanamraj-Radsri/agentic-research-agent.git`
2. **Navigate to the directory**:  
   `cd agentic-research-agent`
3. **Install dependencies**:  
   `npm install`
4. **Run the application**:  
   `npm start`
5. **Open your browser** and go to `http://localhost:3000` to see it in action!

## Prerequisites
- Node.js >= 14.x
- npm (Node Package Manager)
- A web browser (Chrome is recommended)

## Step-by-Step Installation
1. **Install Node.js**:

## Understanding Output
- After you run `npm start`, the server will start and output messages in the terminal.  
- Check the terminal for errors or confirmations about successful connections to databases or services used.

## API Configuration
- You can configure the API endpoints in the `config.js` file found in the root directory.  
- Set your API keys and other necessary environment variables.

## Troubleshooting
- If you encounter problems while running the application, check the following:

  - If you made changes to configuration files, double-check for typos.
